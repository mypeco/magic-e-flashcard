📖 マジックeとは？
マジックeは、英語の重要なフォニックスルールのひとつです。

単語の最後に「e」があると、その前の母音を**アルファベット読み（長母音）**する
最後の「e」自体は読まない

例：

cap → cape（キャップ → ケイプ）
not → note（ナット → ノウト）
bit → bite（ビット → バイト）
